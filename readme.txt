To run this application follow below steps:
1. Make sure pyton 3.8 or above is installed.
2. Install all requirements for requirements.txt file using command --> pip install -r requirements.txt
3. Now to run the FasAPi application run this command --> uvicorn main:app --reload
4. After the application has run successfully you can check the api docs at --> http://hostname/docs or http://hostname/redoc
